## [v0.6.0] - 2026-06-07（PR #11 & PR #12 合并）
### Added
- **项目正式更名为「林中箱庭」**，同步更新页面标题与品牌标识
- 前端整体视觉重构，贴合丛林阳光主题风格，营造林间氛围感
- 采用丛林实景背景 + 光影渐变配色，主色调为自然墨绿/浅绿，适配林间光影感
- 新增半透明毛玻璃卡片、渐变按钮、专属滚动条，界面质感统一协调
- 自定义代码区语法高亮，提升剧本编辑与查看体验
- 重构页面主标题样式，增加半透明木质底色+毛玻璃效果，提升与背景图区分度（PR12）
- 优化下载文件名，自动添加时间戳标识，避免文件覆盖（PR12）

### Changed
- 优化按钮按压状态、输入框焦点高亮，提升交互流畅度（PR12）
- 完善前端异常捕获与防重复提交逻辑，提升系统稳定性（PR12）

### Fixed
- 保留全部原有业务逻辑、交互功能、接口联调，无功能删减

## [v0.5.0] - 2026-06-07（PR #10 合并）
### Added
- 统一 API 密钥配置流程，前端新增密钥输入与保存入口
- 支持自定义密钥与多模型扩展，打通前后端联调
- 修复 Doubao 模型调用失败问题，替换为稳定的 `doubao-seed-1-6-251015` 接入点
- 前端新增模型选择下拉框，为后续多模型扩展预留接口

### Changed
- 保留 `500s` 超时、`temperature=0.3` 等核心模型参数配置

## [v0.4.2] - 2026-06-07（PR #9 合并）

- 优化启动日志格式，增加成功提示与操作说明

## [v0.4.0] - 2026-06-06
### Added
- 新增剧本YAML结构化生成接口 `/api/generate-script`
- 支持自动清理大模型输出的markdown代码块标记
- 内置YAML语法校验，解析失败返回明确错误信息和原始输出

### Fixed
- 统一方法名为 `generate_text()`，解决YAML生成调用不匹配问题
- 修复 `generate_script_yaml` 方法缩进错误，正确归属类内
- 实现标准单例模式，全局仅初始化一次客户端
- 将 `split_text()` 分片函数封装到 `DoubaoClient` 类内
- 规范 `yaml` 库导入位置，符合Python编码规范
- 新增输入前置校验，空文本或纯空格直接抛出异常，节省API调用
